# Information Security Awareness Website

This project is a static educational website about everyday cybersecurity awareness.

It covers:

- password hygiene
- phishing and malicious links
- short URL risk awareness
- practical daily protection tips

The site also includes:

- an embedded explainer video
- a phishing-style interaction demo
- a lightweight browser game to make the learning experience more memorable

## Stack

- HTML
- CSS
- JavaScript

## Project Goal

The goal of this project is to turn security awareness content into a more approachable and interactive learning experience for non-technical users.

## Run Locally

Open `index.html` directly in a browser, or serve the folder with any static file server.
